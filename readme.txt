Структура файлов и папок

index.php - главная страница приложения
style.css - стили для элементов страницы
script.js - поведение элементов страницы
/images - папка для хранения изображений для приложения
/configs - файлы настроек для приложения
/modules - папка содержит функциональные блоки приложения
    get_user.php - функционал для получения данных пользователя и БД
    perepiska.php - функционал для вывода сообщений пользователей
    spisok.php - функционал для вывода пользователей в списке пользователей
